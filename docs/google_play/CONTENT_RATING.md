# Google Play — klasyfikacja treści (Content rating / IARC)

NapStack to aplikacja **timera drzemki / alarmów** bez treści erotycznych, przemocy, hazardu ani czatu społecznościowego.

## Kwestionariusz (typowe odpowiedzi)

Podczas **Play Console → Twoja aplikacja → Polityka aplikacji → Klasyfikacja treści** wypełnij kwestionariusz zgodnie z faktycznymi funkcjami. Poniżej **orientacyjne** odpowiedzi — zawsze potwierdź je przy każdej nowej funkcji (np. czat, treści UGC).

| Temat | Typowa odpowiedź |
|--------|------------------|
| Przemoc | **Brak** lub minimalna (brak scen przemocy w aplikacji) |
| Seks / nagość | **Brak** |
| Wulgaryzmy / wulgarne żarty | **Brak** |
| Substancje uzależniające | **Brak** (aplikacja nie promuje używek) |
| Hazard | **Brak** |
| Strach / horror | **Brak** |
| Użytkownicy mogą wchodzić w interakcje / udostępniać treści | **Nie** (brak forum, czatu, publicznego feedu) |
| Udostępnianie lokalizacji innym użytkownikom | **Nie** |
| Zakupy w aplikacji | **Tak** (NapStack Pro) — kwestionariusz zapyta o monetyzację; wskaż **zakupy cyfrowe** |

## Oczekiwany wynik

Dla podobnych aplikacji „utility / productivity / wellness (non-medical)” często wychodzi **PEGI 3** / **Everyone** (w zależności od regionu i dokładnych pytań IARC). **Ostateczna ikona i opis przyznawane są przez kwestionariusz** — nie traktuj tego pliku jako gwarancji etykiety.

## Spójność ze sklepem

- W opisie nie obiecuj funkcji medycznych (zgodnie z regulaminem: nie jest urządzeniem medycznym).
- Jeśli dodasz treści generowane przez użytkowników lub czat, **ponownie** uruchom klasyfikację.
